詳解

拡張フォント関連.

API をフックすることで、pceFont 系 API に拡張フォントを追加します。¥n 拡張フォントは、AQUAPLUS 製ゲームの仕様に準じる予定です。¥n 

    拡張フォントの文字コード
    - 0xF040 - 「!?」
    - 0xF041 - 「!!」
    - 0xF042 - ハートマーク
    - 0xF046 - ２文字分の「〜」の左側
    - 0xF047 - ２文字分の「〜」の右側

@author zurachu
