説明

フォント設定の代行処理.

pceFontSet 系 API で設定した内容を取得する API が公開されていないため、 API をフックして設定値を保存しておき、取得できるようにします。

警告:
API をフックしない状態で pceFontSet〜() を直接呼び出すと、 設定内容の整合性が取れなくなるので注意して下さい。
